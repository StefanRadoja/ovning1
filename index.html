viktiga html koder

På denna siddda ska jag bygga upp mitt eget refernsverk med viktiga htmlkoder som behöver för att kunna bygga bra webbsidor

HTML Table

Kod Betydelse
<html> Representerar roten till ett HTML-dokument, är en behållare för alla andra HTML-element
<head> Insider html-dokumenten och talar om att här kommer det komam HTML-kod
<title> Definierar dokumentets titel.
<body Definierar dokumentets kropp, innehåller alt innehåll som rubrikerm stycken, bilder hyperlänkar, tabeller, listor, etc.
h1 Rubriknivå1, det finns ner til nivå 6 som är den minsta rubriknivån
p Taggen som omslutar text.
table Definierar en HTML-tabell, innehåller kotumner och rader.
img används för att bädda in en bild på en HTML-sida
form Används för att skapa ett HTML-formulär för användarinmatning.
input Anger aett inmatningsfält där anvöndaren kan ange data.
button Definierar en kickbar knapp.
a Skapar en länk
b Taggen som gör fel text.
q Taggen som markerar ett länge citat. Denna bryts ut och hanteras separat.
